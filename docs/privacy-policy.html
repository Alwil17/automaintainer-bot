Privacy Policy

Effective date: June 17, 2025

AutoMaintainer-bot is a GitHub App designed to automate issue creation and labeling based on TODO comments in your code. We value your privacy and are committed to protecting your data.

1. Data Collection

  • The bot only accesses repository data necessary to perform its functions (scanning for TODO comments, creating issues, and managing labels).
  • No personal user data is collected, stored, or processed.
  • No code or metadata is stored outside of GitHub infrastructure.

2. Data Usage

  • All processing occurs in-memory or within GitHub’s infrastructure.
  • Data is used solely to automate issue creation and labeling in your repositories.
  • No data is transmitted to third parties or external services.

3. Data Retention

  • AutoMaintainer-bot does not persistently store any repository content or user data outside of GitHub.
  • All actions are performed via GitHub’s official APIs and webhooks.

4. Security

  • Authentication is handled using GitHub App credentials and webhooks are validated for authenticity.
  • Private keys and secrets are managed securely and never logged or exposed.
  • The bot follows GitHub security best practices and is regularly updated for security compliance.

5. Compliance

  • Complies with the GitHub Marketplace Developer Agreement.
  • Designed to align with relevant sections of the EU AI Act for transparency and data governance.
  • No automated decision-making or profiling is performed.

6. Contact

For questions or concerns about this privacy policy, please open an issue on our GitHub repository.
